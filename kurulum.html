BetterXP'ı BetterXP Installer kuracaktır.
Burada BetterXP Installer'i çalıştırma talimatları yer almaktadır.

Yükleyici çalıştırma:
  1. Git ile Python3 paketini yüklemek gerekmektedir. Bunun için:
    2. Eğer Debian GNU/Linux tabanlı bir GNU/Linux dağıtımı kullanıyorsanız şu komutu terminale yazın:
    sudo apt install git python3 -y
    Eğer Fedora Linux tabanlı bir GNU/Linux dağıtımı kullanıyorsanız şu komutu terminale yazın:
    sudo dnf install git python3 -y
    Eğer Solus tabanlı bir GNU/Linux dağıtımı kullanıyorsanız şu komutu terminale yazın:
    sudo eopkg it git python3 -y
  2. Terminale şu komutu girin:
    git clone https://github.com/androidprotmmas/BetterXP-Installer.git ; cd BetterXP-Installer ; sudo python3 installer.py

Artık BetterXP Installer açılacak ve bilgisayarınıza sizin yönlerdirmenize bağlı olarak BetterXP'i kuracaktır.
